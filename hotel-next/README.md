# Memo — vertical pages (Next.js)

See **[HOTEL_PAGE_REFACTOR.md](./HOTEL_PAGE_REFACTOR.md)** for structure, source mapping, and how to run the dev server (hotel + datacenter).

Quick start: `cd hotel-next` → `npm install` → `npm run dev` → open **http://localhost:3000/hotel**, **http://localhost:3000/datacenter**, or **http://localhost:3000/warehouse** (include **`:3000`** — plain `http://localhost/datacenter` without the Next server causes unstyled pages; see troubleshooting in `HOTEL_PAGE_REFACTOR.md`).
